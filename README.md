## 1. Git 초기 설정 (Initial Setup)
Git을 처음 설치한 후 한 번만 설정하면 됩니다.

### 버전 및 사용자 정보 설정
```bash
# Git 버전 확인
git --version

# 사용자 이름 설정 (커밋 기록용)
git config --global user.name "이름"

# 사용자 이메일 설정 (커밋 기록용)
git config --global user.email "xxx@xxx.com"

# 설정 확인
git config user.name
git config user.email
